# ChatGPT Clone - Frontend Project

This repository contains the complete project documentation and source code for a high-fidelity ChatGPT frontend clone built with React.

## 📋 Project Overview

A modern, interactive chat interface that replicates all major ChatGPT features including:
- 💬 Streaming text responses (character-by-character animation)
- ✍️ Rich text editor (bold, italic, code, emoji)
- 📂 Conversation sidebar (history, rename, delete)
- 📝 Markdown rendering + code syntax highlighting
- 🎬 Copy, regenerate, and stop generation actions
- 🌙 Dark mode toggle
- 📱 Mobile responsive design
- 💾 localStorage persistence

**Backend:** Hardcoded mock JSON responses (no real API)

## 🏗️ Tech Stack

- **Frontend:** React 18+
- **State Management:** Context API or Zustand
- **Rich Text Editor:** Lexical (React-first)
- **Markdown:** react-markdown + remark-gfm
- **Syntax Highlighting:** Prism.js or Highlight.js
- **Styling:** Tailwind CSS
- **Build:** Vite
- **Deployment:** Vercel
